# **25WSA032 Coursework Code Quality**

# **Introduction**
In elements of this coursework you are expected to write code to solve specific problems, and your code will be evaluated. Note however, code quality is not the dominant component of these assessments so please tailor your effort in the time you have available to all the assessment criteria given in the task.

## Robust Error Handling and Defensive Programming
Where appropriate :
- The program should include mechanisms to **anticipate, detect and handle invalid data**.
 Clear error reporting or exception handling logic should be included.

## Efficient Use of Memory and Computation
- Where approriate the code should demonstrate awareness of **embedded system limitations and techniques**, including:
    - Efficient use of RAM
    - Avoidance of unnecessary data duplication
    - Efficient implementation of loops and calculations
    - Appropriate use of **pointers or references** where beneficial (e.g., passing arrays efficiently).
  - Use of appropriate **data types and memory structures** to reduce computational overhead.

## Effective use of functions and modular code design
- to improve readability and code reuse/maintainability.

## Readability and Clarity
- Variables and functions should be **meaningfully named**.
- Inclusion of **clear comments explaining the purpose of key sections**.
- The program structure should allow another developer to **easily understand the algorithm and system logic** (readability over slickness).
- In Python the use of doc strings for functions, modules, classes and methods

## Program Reliability
- An sketch for Arduino should compile and run correctly on the Arduino platform.
- A program should demonstrate **stable operation over the normal expected runtime** on the intended platform.
- The behaviour of the algorithm should match the intended design requirements.

## Innovation and Engineering Insight 
- Demonstration of **creative or advanced approaches** to program design, optimisation, or data/signal processing
- Implementation of **additional efficiency improvements or enhancements** beyond the basic requirements.
